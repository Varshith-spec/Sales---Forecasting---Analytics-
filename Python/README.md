This folder contains the Python code used for the Sales Forecasting project.

The Python scripts perform the following tasks:

• Data cleaning and preprocessing using Pandas
• Exploratory Data Analysis (EDA)
• Sales trend analysis and visualization using Matplotlib
• Time-series forecasting using Prophet / ARIMA
• Preparing datasets for dashboard visualization

Libraries used:
- Pandas
- NumPy
- Matplotlib
- Prophet
- Scikit-learn
